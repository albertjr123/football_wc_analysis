# FIFA World Cup Data Analysis (2002–2022)
A comprehensive data analysis project exploring FIFA World Cup tournaments from 2002 to 2022 using Python and data visualization techniques.

## Overview
This project analyzes FIFA World Cup data across five tournaments (2002, 2006, 2010, 2014, 2018, 2022) to uncover insights about:

- Team performance trends
- Player goal statistics
- Award winners across tournaments
- Goal distribution patterns (penalties, own goals, open play)

 ## Objectives

- Clean and preprocess multi-tournament football datasets
- Analyze team-level performance across World Cups
- Identify top players and goal scorers
- Study distribution of goals by type
- Compare award winners across tournaments
